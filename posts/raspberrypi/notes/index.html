    Terminal multiplexer⚓︎

    screen is a terminal multiplexer program that allows managing several Terminal sessions from with the single window. You can start a new session and detach it to leave it running in background even your terminal is ended (like over ssh), then you can resume that session later.

    Similar: tmux

    CPU Usage⚓︎

    top shows CPU Usage in a period of time which is the update interval.

    ps shows average CPU Usage of a process from when it has started.

    tload to see CPU Usage history in simple graph in terminal.

    Python⚓︎

    Most packages can be install using sudo apt-get install followed by python-<packagename> (for Python2) or python3-<packagename>.

    In some cases, you can't do that, you still can install packages via pip. Install pip first:

    sudo apt install python-pip python3-pip
